How can the same content feel completely different through design?

This project explores how layout, spacing, typography, image, motion, and color can change the emotional experience of a website. The content stays connected, but each style page creates a different mood and rhythm.

01

About the Project

One project, different emotional systems.

My final project focuses on visual mood in web design. I am interested in how the same idea can feel quiet, distant, personal, or nostalgic depending on the design system around it.

Instead of treating a webpage as only information, I treat it as an atmosphere. The viewer moves through spacing, color, motion, and interaction as part of the emotional experience.

02

Design Concept

Design choices become emotional choices.

Calm

The calm page uses large images, slow transitions, soft movement, and empty space to create a quiet visual experience.

Nostalgic

The nostalgic page uses an old Macintosh style, pixel texture, draggable windows, and image fragments to feel more personal and remembered.

Structure

The main page works like a darker index page. It organizes the project and lets the two style pages become the main focus.
